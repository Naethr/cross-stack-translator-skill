# AGENTS.md

This file provides repository-level instructions for AI coding agents working on this project.

This repository is an Agent Skill authoring project. The product is the skill itself, not an application, CLI, backend, VS Code extension, frontend, API, package, runtime scanner, or code modification engine.

## Maintenance instructions

* Keep changes minimal, focused, and easy to review.
* Preserve the instruction-only nature of the project.
* Do not add runtime code unless explicitly requested.
* Do not add dependencies unless explicitly requested.
* Do not broaden the product scope without explicit instruction.
* Prefer precise skill instructions over generic documentation.
* Preserve the distinction between verified facts, assumptions, and unknowns.
* Keep the skill focused on translating an unfamiliar repository through a stack the user already knows.
* Do not turn this project into a generic codebase onboarding skill.
* Do not turn this project into a code reviewer, repository summarizer, documentation generator, or code generator.
* Do not make unsupported claims about skill.sh, Codex, VS Code, marketplaces, or framework support.
* When modifying the skill, keep the output format stable.
